# Expense Tracker 
This is a command line expense tracker application which is made in python

## What it does
This application does the Following:
1. Add, Remove and Delete expenses
2. View your expenses
3. Finding total of the amount

## Features of the Application
- Add an expense with name and amount
- View all expenses
- View total spending
- Delete an expense
- Data persists using CSV storage


## What i got to learn while making this:
- Python lists and dictionaries
- File handling with CSV module
- Try/except for error handling
- Loops and conditionals

## How to run
1. Make sure Python is installed
2. Clone this repository
3. Run this command:
python expense_tracker.py
